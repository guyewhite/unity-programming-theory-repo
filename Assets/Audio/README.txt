AUDIO SETUP INSTRUCTIONS:

1. Add any .wav, .mp3, or .ogg audio file to this folder for the catch sound
2. In Unity, assign the audio file to the GameManager's "Catch Sound" field
3. The game will play this sound whenever a ball is caught

You can find free sound effects at:
- freesound.org
- zapsplat.com
- soundbible.com

Or use Unity's built-in audio generation tools.
